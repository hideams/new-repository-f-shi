# Architect Engineer Platform

A comprehensive web tool for architects and engineers to facilitate their work.

## Features
- Design tools with canvas for drawing plans and models.
- Calculators for areas and costs.
- Resources and references.
- Project management.

## How to Use
Open index.html in your browser to get started.

## Development
The project can be expanded by adding libraries like Three.js for 3D models or React for more complex interfaces.
