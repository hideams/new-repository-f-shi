# Architect Engineer Platform

A professional front-end prototype for an architectural engineering project management platform. This version includes a dashboard, project management cards, task board, Saudi Building Code compliance checklist, budget tracker, team view, and report previews.

## Features
- Executive summary dashboard with active projects, pending tasks, compliance alerts, and budget utilization.
- Project cards with timeline, budget, location, and completion progress.
- Kanban-style task board for status tracking.
- Compliance checklist for SBC-related requirements.
- Budget table with category breakdown and variance calculation.
- Team and report overview sections.
- Light/dark theme toggle.
- Excel export placeholder for project summary.

## Launch
1. Open `index.html` in a browser.
2. Use the sidebar to navigate between dashboard, projects, tasks, compliance, budget, team, and reports.
3. Click `New Project` or `New Task` to add sample items.
4. Click `Export Summary` to download a CSV summary file.

## Notes
- This prototype is a front-end-only implementation in HTML, CSS, and JavaScript.
- Future enhancements can include React, a back-end API, Excel/XLSX integration, file uploads, and real-time notifications.
